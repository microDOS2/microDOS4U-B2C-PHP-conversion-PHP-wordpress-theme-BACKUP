# microDOS(2) Affiliate Creatives - Setup Cheat Sheet

## Step 1: Create Categories

Go to **AffiliateWP > Creatives > Manage Categories** and create:

| Category Name | Description |
|---------------|-------------|
| **Banners** | Image-based banner ads for websites and sidebars |
| **Text Links** | Pre-written promotional copy with embedded affiliate tracking |
| **Social Media** | Square and landscape banners for Instagram, Facebook, etc. |

---

## Step 2: Upload Images to WordPress Media Library

1. Go to **Media > Add New** in WordPress admin
2. Upload all 6 banner images from `/images/creatives/` folder
3. Copy the full URL of each uploaded image
